Oi! eu sou o Rafa

Apaixonado pelo desenvolvimento web desde que entrei na faculdade, brasileiro e nascido em 1998. Sou estudante de Análise e Desenvolvimento de Sistemas na PUC Minas. Sou uma pessoa que adora desafios, assim como todo desenvolvedor, mas em especial aqueles que me tiram da minha zona de conforto, pois as minhas maiores evoluções e aprendizados vierem de momentos em que me vi obrigado a sair dela. Estudante assíduo, que aprecia tomar café e nas horas vagas gosta de jogar uns joguinhos online.

Histórico Acadêmico e Experiências

No primeiro semestre do meu curso, participei ativamente do desenvolvimento de um projeto web front-end que se destacou não apenas pelo seu design atraente, mas também pela experiência do usuário excepcional que proporcionou. Utilizando as linguagens HTML, CSS e JavaScript, conseguimos criar uma plataforma que não apenas atendeu às necessidades dos usuários, mas também recebeu o reconhecimento da faculdade, sendo agraciado com o certificado de projeto destaque do semestre.

No segundo período, embarquei em um projeto mais abrangente, onde trabalhei com uma equipe na criação de uma aplicação ASP.NET Core MVC. Nesse projeto, implementamos um sistema de gerenciamento de cursos e alunos, aproveitando a potência do SQL para garantir um fluxo eficiente de dados. Essa experiência reforçou minha compreensão das estruturas de desenvolvimento back-end e da importância de uma base sólida de dados para aplicações robustas.

No terceiro período, mergulhei no mundo do desenvolvimento móvel ao participar do desenvolvimento de um aplicativo utilizando o framework React Native. Com JavaScript como nosso trampolim, criamos um sebo online com uma interface intuitiva e funcionalidades avançadas. Essa incursão no desenvolvimento mobile expandiu meus horizontes e me permitiu compreender a importância de otimizar aplicativos para uma variedade de dispositivos.

Neste ponto da minha jornada acadêmica, estou imerso em um projeto empolgante que envolve o desenvolvimento simultâneo de uma aplicação web e móvel. O desafio adicional aqui é criar uma Web API que permita a interação perfeita entre as duas plataformas. Estou empolgado em aplicar todos os conhecimentos adquiridos até agora para garantir que essas duas interfaces se complementem perfeitamente, oferecendo aos usuários uma experiência perfeita e integrada.
